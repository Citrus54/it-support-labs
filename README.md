# IT Support Labs

Hands-on IT support and troubleshooting lab portfolio focused on Windows Server, Active Directory, DNS, networking, file share permissions, and SaaS/web support scenarios.

## Purpose
This repository documents practical support labs built to simulate real-world incidents, diagnose root causes, apply fixes, and verify resolution.

## Skills Demonstrated
- Active Directory user and group management
- DNS troubleshooting
- Windows networking fundamentals
- File share and permissions troubleshooting
- SaaS/web support troubleshooting
- Windows system administration
- Ticket-style documentation and resolution notes

## Lab Documentation Format
Each lab follows a consistent incident-response structure:

- Issue
- Environment
- Symptoms
- Tools Used
- Troubleshooting Steps
- Root Cause
- Resolution
- Verification
- Key Takeaway
